Asuka Pet Assistant v0.3.0 Preview

这是一个 Windows 桌面宠物预览版。

启动方式：
双击 Asuka Pet Assistant-0.3.0-portable.exe 即可运行。

基础操作：
- 单击桌宠：挥手
- 双击桌宠：跳一下
- 按住或拖拽：拎起/拖动桌宠
- 快速连点：触发生气反馈
- 右键桌宠：打开控制菜单
- 托盘图标：隐藏、显示、退出

右键菜单功能：
- 挥手、跳一下、查看、生气
- 向左走、向右走
- 提醒我
- 设置
- 对话
- 小组件
- 隐藏到托盘
- 退出

AI 对话：
如果要使用 AI 对话，请先在设置里填写 API 信息。
不填写 API 时，桌宠基础互动仍可正常使用。

提醒功能：
右键菜单选择“提醒我”，输入内容和分钟数。
到时间后桌宠会弹出提醒。

说明：
这是 Preview 版，主要用于体验和测试。
动作、台词、设置和 AI 能力后续还会继续优化。
