Asuka Pet Assistant v0.3.0 Preview 更新说明

本版定位：
第一个可对外试用的 Windows 桌面宠物 Preview 版。

主要内容：
- 完成透明置顶桌宠窗口
- 支持单击、双击、拖拽、快速连点、右键菜单
- 支持挥手、跳跃、查看、生气、走动、拖拽等动作
- 修复向左走动画撕裂问题
- 优化右键菜单 UI
- 增加系统托盘隐藏/显示/退出
- 增加设置持久化
- 增加 AI 对话入口和 API 设置
- 增加中文台词气泡
- 增加提醒我功能
- 增加时间段感知台词
- 打包为 Windows portable exe

已知限制：
- 这是预览版，不是最终稳定版
- AI 对话需要用户自行配置 API
- 部分动画和台词后续仍会继续打磨
- 目前只提供 Windows 便携版

推荐测试：
- 双击 exe 启动
- 右键菜单是否正常
- 单击/双击/拖拽/快速连点是否正常
- 提醒功能是否正常
- 对话框打开/关闭是否正常
- 托盘隐藏/显示/退出是否正常
