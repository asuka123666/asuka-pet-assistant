Asuka Pet Assistant v0.5.1 Preview

这是一个 Windows 桌面宠物预览版。

启动方式：
双击 Asuka Pet Assistant-0.5.1-portable.exe 即可运行。
如果已经有桌宠在运行，再次打开会唤起现有桌宠，不会重复启动。

基础操作：
- 单击桌宠：挥手
- 双击桌宠：跳一下
- 按住或拖拽：拎起/拖动桌宠
- 快速连点：触发生气反馈
- 靠近屏幕边缘或鼠标靠近：触发额外小动作
- 右键桌宠：打开控制菜单
- 托盘图标：隐藏、显示、退出

右键菜单功能：
- 挥手、跳一下、查看、生气
- 向左走、向右走
- 提醒我
- 设置
- 对话
- 小组件
- 资源状态卡
- 隐藏到托盘
- 退出

资源小球：
资源占用会收纳成桌宠角落里的液体小球，点击后展开。
展开后可以查看 CPU、内存、系统盘和 GPU 占用。
可以在设置里调整小球大小、透明度、位置、百分比文字、GPU 监控和高占用提醒。
现在也可以直接拖动收起状态的小球来放到喜欢的位置，位置会自动记住。
GPU 一行会在可读取时显示独显/核显、占用和显存状态；读取不到时会安静显示，不弹错误。

AI 对话：
如果要使用 AI 对话，请先在设置里填写 API 信息。
不填写 API 时，桌宠基础互动仍可正常使用。

提醒功能：
右键菜单选择“提醒我”，输入内容和分钟数。
到时间后桌宠会弹出提醒。

说明：
这是 Preview 版，主要用于体验和测试。
动作、台词、设置和 AI 能力后续还会继续优化。
